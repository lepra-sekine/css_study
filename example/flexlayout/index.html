CSSのサイズ指定

ブロック要素の中で子要素の幅・高さをどう扱うかを、実例を通じてわかりやすく学べる教材です。

親要素に display: flex; を指定すると、子要素がフレックスコンテナ内で横並びになります。子要素に width と height を指定していない場合、フレックスのデフォルト動作に従い、利用可能なスペースを均等に分配します。

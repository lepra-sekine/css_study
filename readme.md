# CSS苦手意識をなくす場所

[実装例プレビュー](https://lepra-sekine.github.io/css_study/index.html)

## 本リポジトリの目的

このリポジトリは「CSSの苦手意識をなくす場所」をめざす、ゆるい学習スペースです。

- CSSの基礎がなんとなくわかってきたけど、もう一歩進みたい初級〜中級者が対象。
- 「今までなんとなく書いていたけど、本当はどういう設計にすべきか」を実践で学べるように。
- 実装例（OKパターン）だけでなく、やりがちなアンチパターンも直接見て学べる形にします。
- 難しすぎず、堅苦しくない言葉で進めつつ、実務で活きる知識を届けたいです。

## 本リポジトリのスコープ

- CSS全般
    - 基本の文法、レイアウト、色・フォント、アニメーション、レスポンシブ対応などの実例。
    - 設計思想としてはOOCSS（構造と見た目を分ける）をベースにして、読みやすくメンテしやすい書き方を紹介。

#### 趣旨とボリュームの関係で本リポジトリではあえて触れない項目

- Sass
- ReactやVue.jsなどのJavaScriptフレームワーク
